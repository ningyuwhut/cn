今天写程序的时候遇到一个很诡异的问题,代码逻辑如下:

    String a="";
    String b="";
    if( a == "foo" )
        ...
    if( b == "bar" )
	...

很显然,这是错误的.java中比较字符串是否相等应该使用`equals`方法,而不是使用`==`.

两者的区别是equals比较字符串对象的内容是否相等,而==比较两个字符串是否指向同一个对象.
之前也遇到过这个问题, 时间一长又忘了,浪费了很长时间调试,不应该.
